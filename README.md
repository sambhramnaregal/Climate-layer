# Climate Layer

Climate Layer is a Flask-based weather application that provides real-time weather data and forecasts using the [Open-Meteo API](https://open-meteo.com/).

## Features

-   **Current Weather**: Displays current temperature, wind speed/direction, humidity, and precipitation.
-   **Hourly Forecast**: Shows hourly direct radiation data for the next 24 hours.
-   **Location Details**: Provides elevation and UTC offset information for the queried location.
-   **Interactive UI**: Simple web interface to input latitude and longitude coordinates.

## Tech Stack

-   **Backend**: Python, Flask
-   **API**: Open-Meteo (via `openmeteo-requests` and `pandas`)
-   **Frontend**: HTML, CSS, JavaScript

## Prerequisites

-   Python 3.x
-   pip

## Installation

1.  Clone the repository or navigate to the project directory.

2.  Create a virtual environment (optional but recommended):
    ```bash
    python -m venv venv
    # On Windows:
    .\venv\Scripts\activate
    # On macOS/Linux:
    source venv/bin/activate
    ```

3.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Running the Application

1.  Start the Flask server:
    ```bash
    python app.py
    ```

2.  Open your web browser and navigate to:
    ```
    http://127.0.0.1:5001/
    ```

3.  Enter the Latitude and Longitude coordinates (e.g., Lat: 52.52, Lon: 13.41 for Berlin) and click "Fetch Data".

## Project Structure

-   `app.py`: Main Flask application file.
-   `temperature.py`: Logic for fetching and processing weather data from Open-Meteo.
-   `.cache.sqlite`: A local cache file generated by the application to store weather API responses for faster load times. (This file is automatically created when you run the app).
-   `static/`: Contains CSS and JavaScript files.
-   `templates/`: Contains HTML templates.
-   `requirements.txt`: Python dependencies.
